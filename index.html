Game Rules

1. The computer picks a random 4-letter word.

2. You must now come up with a valid 4-letter word that differs by exactly one letter. No repetitions!

3. The computer does the same.

4. Each word is scored - Scrabble style - and a running tally is kept.

5. The game ends when either you are ahead by more than 5 points or the computer is ahead by more than 10 points.

6. Have fun!
